# 프롬프트 ① — AI 독해 도우미 웹앱 구현

아래 내용을 Claude Code에 붙여넣으세요.

---

당신은 풀스택 웹 개발자입니다. 직업계고 1학년 학생용 **AI 독해 도우미 웹앱**을 완성해 주세요.
이 저장소에는 이미 환경 세팅(`package.json`, `vercel.json`, `.env.example`, 스텁 `public/index.html`, `api/analyze.js`)이 되어 있습니다. 스텁을 실제 동작 코드로 채우면 됩니다.

## 무엇을 만드나
학생이 **디자인 직무 관련 글을 붙여넣고 [분석하기]를 누르면**, 다음 네 가지를 실시간으로 생성해 보여 주는 웹앱:
1. **어려운 어휘 풀이** — 글 속 어려운 낱말/전문용어를 골라 직업계고 1학년 눈높이의 쉬운 뜻으로
2. **문단별 요약** — 문단마다 한두 문장 요약
3. **핵심어 표시** — 글에서 중요한 핵심어 목록 (가능하면 원문 영역에서 하이라이트)
4. **이해 점검 질문** — 글 내용 확인 질문 3~5개 + 모범답안 + 힌트(처음엔 답을 가려 두고, 학생이 누르면 펼침)

## 기술 요구사항
- **아키텍처**: 정적 프런트엔드(`public/index.html`, 바닐라 JS) + 서버리스 백엔드(`api/analyze.js`). 같은 도메인이므로 CORS 불필요.
- **보안(중요)**: OpenAI 키는 `process.env.OPENAI_API_KEY` 로 **백엔드에서만** 사용. 절대 프런트엔드 코드/네트워크 응답에 키를 노출하지 말 것.
- **API**: OpenAI Chat Completions (`https://api.openai.com/v1/chat/completions`), 헤더 `Authorization: Bearer ${OPENAI_API_KEY}`.
  - 모델: `process.env.OPENAI_MODEL` (없으면 `"gpt-4o-mini"`).
  - `response_format: { type: "json_object" }` 로 구조화 출력. 시스템 메시지에 "직업계고 1학년이 이해할 쉬운 한국어로, 반드시 아래 JSON 형식으로만 답하라"고 지시.
- **요청/응답 계약** (스텁 주석과 동일하게):
  - 요청: `POST /api/analyze` body `{ "text": "..." }`
  - 응답: `{ vocab:[{word,meaning}], summaries:[{paragraph,summary}], keywords:[string], questions:[{q,answer,hint}] }`
  - 입력 검증: 빈 값/공백/너무 긴 입력(예: 8,000자 초과) 거르고 친절한 오류 메시지.
- **한 번의 요청으로 네 가지를 모두** 받아 비용을 아낄 것 (기능별 개별 호출 금지).

## UI/UX 요구사항 (학생 친화)
- 한국어 UI, **큰 글씨**, 크롬북·태블릿·모바일 모두 보기 좋게(반응형). 외부 의존은 CDN 한 개 이내(예: 폰트)로 최소화.
- 화면: 상단 제목/간단 안내 → 왼쪽 '원문' 영역(붙여넣기 textarea) → [분석하기] 버튼 → 오른쪽(또는 아래) '결과' 영역에 4개 카드(어휘/요약/핵심어/질문).
- 분석 중 **로딩 표시**, 실패 시 **알기 쉬운 오류 메시지**.
- 질문 카드: 모범답안은 기본 숨김 → [답 보기] 토글.
- 상단에 안내 문구: "이름·연락처 같은 개인정보는 넣지 마세요. AI 답은 틀릴 수 있으니 원문과 비교하세요."
