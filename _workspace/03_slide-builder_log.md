# 03 slide-builder 작업 로그

## 산출물
- 스크립트: `materials/make_slides.py`
- 산출물: `materials/output/수업PPT_디자인직무읽기.pptx` (16:9, 11장, 약 46KB)

## 생성 성공 여부
**성공** — 11장 모두 생성 확인. 16:9(13.333×7.5인치) / 텍스트 run 78개 전부에 eastAsia·라틴 '맑은 고딕' 지정(한글 깨짐 방지) / 네이비·블루 단일 톤 / 4번 슬라이드 QR placeholder(점선 회색 사각형 + "여기에 웹앱 주소·QR를 넣어 주세요" + 주소 입력선) 포함.

## 슬라이드 개요 (제목 줄, 명세 03_PPT.md 순서 준수)
1. AI 독해 도우미로 디자인 직무 글 읽기 (표지)
2. 디자인 현장의 '읽기' (동기유발: 트렌드 리포트·브리프·지시서·설명서 → 잘 만들려면 잘 읽어야 한다)
3. 오늘의 목표 (핵심·의도 파악 + AI로 읽기 과정 점검)
4. AI 독해 도우미 소개 (어휘·요약·핵심어·점검질문 4기능 + 사용법 3단계 + QR/웹앱 주소 자리)
5. AI 사용 주의점 (①개인정보 금지 ②틀릴 수 있다·원문 대조 ③보조 도구로만 ④베끼지 말기)
6. 활동 1: 스스로 읽기
7. 활동 2: AI 도우미로 비계 받기 (어휘→요약·핵심어→점검질문)
8. 활동 3: 핵심 정리 (중심내용·글쓴이 의도·직무 적용)
9. 활동 4: 짝·모둠 점검
10. 정리 (한 문장 요약 발표 + AI 도움/한계 소감)
11. 다음 시간 (트렌드 분석 → 분야별 콘셉트 기획·발표)

AI 윤리 3원칙(개인정보 금지·원문 대조·보조 도구로만/내 말로)은 5번에 4개 카드로 집약하고 활동 2·3 슬라이드에도 일관 반영함.

## 실행 방법
`python materials/make_slides.py` (Windows에서는 슬래시 경로 사용; 백슬래시 `materials\make_slides.py`는 셸 이스케이프 이슈로 실패할 수 있음)
