# Claude Code 작업 시작 안내

이 폴더는 **직업계고 1학년 「국어 × 디자인」 읽기 독해 수업**을 위한 산출물을 만드는 작업 공간입니다.
실제 구현은 Claude Code로 진행하세요. 아래 프롬프트를 **순서대로** 복사해 Claude Code에 붙여넣으면 됩니다.

## 작업 순서
1. `01_웹앱.md` — AI 독해 도우미 웹앱 (OpenAI GPT API, 실시간)
2. `02_활동지.md` — 학생용 인쇄 활동지 (.docx)
3. `03_PPT.md` — 수업용 슬라이드 (.pptx)

## 공통 맥락 (세 프롬프트가 공유하는 배경 — 그대로 참고)
- **프로그램**: 2026 AI 징검다리 — 기초학력·직업 문제해결·SEL 통합 역량 강화
- **대상**: 직업계고 1학년 (중3→직업계고 전환기 학생). 읽기 수준차가 큼.
- **수업 목표**: 디자인 직무 관련 글(트렌드 리포트, 클라이언트 브리프, 작업 지시서, 소재·공정 설명서 등)을
  읽고 **핵심 정보와 글쓴이의 의도**를 파악하며, AI 독해 도우미로 **자신의 읽기 과정을 점검·조정**한다.
- **국어 성취기준**: [10공국1-02-03] 진로·관심 분야 글을 주제 통합적으로 읽고 읽기 과정을 점검·조정한다.
- **톤**: 직업계고 1학년 눈높이의 **쉬운 한국어**, 격려하는 말투. 어려운 한자어·전문용어는 풀어서.
- **AI 윤리/개인정보**: 학생 이름 등 개인정보 입력 금지, AI가 틀릴 수 있으니 원문과 대조, '보조 도구'로만.

## 환경(이미 세팅됨)
- 웹앱: Node 18+ / Vercel 배포 가정. 프런트엔드 `public/`, 백엔드 `api/`.
- 키 보안: OpenAI 키는 `.env`(예시는 `.env.example`)에만 두고 **백엔드에서만** 사용. 프런트엔드로 노출 금지.
- 문서: `materials/`에서 작업. 파이썬 라이브러리는 `materials/requirements.txt` 참고.

## 배포·테스트(웹앱)
1. `.env.example` → `.env` 로 복사 후 실제 OpenAI 키 입력
2. `npx vercel dev` 로 로컬 테스트 (http://localhost:3000)
3. `npx vercel` 로 배포 → Vercel 대시보드 환경변수에 `OPENAI_API_KEY`, `OPENAI_MODEL` 등록
4. 배포 URL을 학생에게 QR/링크로 공유
