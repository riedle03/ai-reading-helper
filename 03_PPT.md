# 프롬프트 ③ — 수업용 PPT(.pptx) 만들기

아래 내용을 Claude Code에 붙여넣으세요.

---

직업계고 1학년 「국어 × 디자인」 읽기 독해 수업에서 쓸 **수업용 슬라이드(.pptx)**를 만들어 주세요.
`python-pptx`로 생성합니다(`materials/requirements.txt` 참고, 없으면 `pip install python-pptx`).
생성 스크립트는 `materials/make_slides.py`, 출력은 `materials/output/수업PPT_디자인직무읽기.pptx` 로 저장하세요.

## 슬라이드 구성(11장, 16:9)
1. **표지** — 「AI 독해 도우미로 디자인 직무 글 읽기」 / 부제: 직업계고 1학년 · 국어 × 디자인
2. **디자인 현장의 ‘읽기’** (동기유발) — 디자이너가 실제로 읽는 글: 트렌드 리포트, 클라이언트 브리프, 작업 지시서, 소재·공정 설명서 → "잘 만들려면 잘 읽어야 한다"
3. **오늘의 목표** — 디자인 직무 글을 읽고 핵심·의도를 파악하고, AI 독해 도우미로 읽기 과정을 점검할 수 있다
4. **AI 독해 도우미 소개** — 어휘 풀이·문단 요약·핵심어·이해 점검 질문 / 사용법 3단계 / 웹앱 주소·QR 넣을 자리(placeholder)
5. **AI 사용 주의점** — ① 개인정보 입력 금지 ② AI는 틀릴 수 있다(원문과 대조) ③ '보조 도구'로만 ④ 베끼지 말고 내 말로
6. **활동 1: 스스로 읽기** — 도움 없이 끝까지, 모르는 부분 표시
7. **활동 2: AI 도우미로 비계 받기** — 어휘 확인 → 문단 요약·핵심어 → 점검 질문 풀기
8. **활동 3: 핵심 정리** — 중심 내용 / 글쓴이 의도 / 디자인 직무 적용
9. **활동 4: 짝·모둠 점검** — 정리한 내용 비교, 다르게 이해한 부분 토의
10. **정리** — 한 문장 요약 발표 + 소감(AI 도우미의 도움/한계)
11. **다음 시간** — 읽은 자료로 디자인 트렌드 분석 → 직업 분야별 콘셉트 기획·발표

## 디자인 요구사항
- 16:9, 한국어 글꼴(맑은 고딕 등) 지정해 **한글 안 깨지게**.
- 슬라이드당 글머리 3~5개 이내, **큰 글씨**, 여백 넉넉히(뒤에서도 잘 보이게).
- 차분하고 깔끔한 단일 톤(예: 네이비/블루 계열 포인트). 과한 장식 X.
- 제목·본문 위계가 분명하게. 활동 슬라이드는 번호/단계가 한눈에.
- 4번 슬라이드에는 웹앱 주소/QR를 넣을 빈 자리와 안내 텍스트 표시.
